# JAJA-SPANISH <깔깔스페인어>

깔깔스페인어는 세계 21개국에서 약 4억명이 사용하는 스페인어를 원어민 선생님들과 1:1 화상으로 배우는 교육 플랫폼 입니다.

## DEV

### Folder Structure

```
- src
  - components
    - atoms
    - molecules
    - organims
    - templates
  - containers
  - models
  - pages
  - App.tsx
  - Index.tsx
```

#### `components`

Presentational Components로 Atomic design을 따른다.

#### `containers`

Container Components로 주로 로직, data fetching을 담당한다.

#### `pages`

최종 페이지 뷰

#### `App.tsx`

이곳에서 라우팅이 일어난다.

### Commit Rule

#### Commit messages

-   "[Emoji] Messages "

#### Emojis

-   :package: [Add or update packages]
-   :sparkles: [Introduce new features]
-   :hammer: [Add or update development scripts]
-   :bug: [Fix a bug]
-   :nail_care: [Update the UI and styles, CSS]
-   :rocket: [Deploy stuff]
-   :memo: [Write docs]
-   :white_check_mark: [Add tests]
-   :art: [Improve structure]
